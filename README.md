## Sobre 
O projeto é uma lista de tarefas, conhecida como ToDo List, que tem como funcionalidades:
- Inserir itens
- Marcar item como concluído
- Remover item 
- Salvar itens mesmo após recarregamento (LocalStorage)

## Tecnologias usadas:
- HTML5
- CSS3
- Sass
- JavaScript
